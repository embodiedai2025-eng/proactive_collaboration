# Action Workflows

These examples show complete API flows that combine helper modules with direct HTTP calls.

- `example_pull_then_pick.py`: select scene, set up robots, move to a blocker, jointly pull it, pick a target object, and place it.

Run from the parent directory:

```bash
python dynateamthor_api/examples/action_workflows/example_pull_then_pick.py --base-url http://127.0.0.1:1212/
```
